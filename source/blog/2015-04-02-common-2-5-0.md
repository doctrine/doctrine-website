We are happy to announce the immediate availability of Doctrine Common
\`2.5.0\`.

Installation
============

You can install this version of Doctrine Common by using Composer and
the following `composer.json` contents:

~~~~ {.sourceCode .json}
{
    "require": {
        "doctrine/common": "2.5.0"
    }
}
~~~~
